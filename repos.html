List of repositories

Here follows a list of research software projects developed within the Heidelberg Collaboratory for Image Processing (HCI).

Click on the icon for more information on the individual projects. Some of these projects are hosted on the GitLab instance of the HCI: their home pages can be reached by clicking either on the project name or on the icon, and the latest version of the code can always be downloaded directly by clicking on the icon. All projects are collectively grouped under the GitLab research_code group. You will need an HCI LDAP account in order to access these projects, please contact Dominic Spangenberger.

External project pages (e.g., on GitHub or BitBucket) are reachable via the icon. Bibliographical references are listed when available.
